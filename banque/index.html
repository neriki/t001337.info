Jeu de banque pour ZX81

Le premier cover disque de l'histoire

Dans le numéro 31 de Micro Systèmes (Mai 83) est apparu un étrange disque audio souple. Celui, lisible sur un lecteur de disques microsillons, contenait un programme pour le ZX81 de Sinclair. C'était le premier cover disk (disque de donn?es informatiques fournis avec un magazine) de l'histoire.
